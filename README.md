# Dragon12Calc
This program was developed for an Embedded Systems class, and must be built using GNU Compiler Collection, or similar software.
Once compiled, the 000.s19 file can be loaded into TeraTerm to use on the board.


All Libraries provided by: Prof. Navid Mohagegh

Known Issues:
- Calculator is not capable of subtracting/dividing
- Calculator is unable to handle any values over 10000: unclear if it's a processor or code
